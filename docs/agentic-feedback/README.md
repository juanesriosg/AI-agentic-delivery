# Agentic Feedback Store

This directory stores durable, reviewable feedback artifacts used by the self-improvement loop.

Runtime files under `.agent/feedback/` are ignored by git. When feedback should teach future agents, capture or promote it here through a self-improvement PR.

Recommended sources:

- manager review comments
- Codex PR review output
- QA and PM findings
- CI failures
- user testing notes
- release retrospectives
- data-analysis findings
- incident/postmortem notes

Do not store secrets, raw PII, credentials, private customer data, or sensitive logs here.
