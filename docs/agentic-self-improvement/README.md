# Agentic Self-Improvement Artifacts

Self-improvement proposals write reviewable artifacts here.

Expected files for a proposal:

```text
docs/agentic-self-improvement/proposals/<proposal-id>.md
docs/agentic-self-improvement/feedback-bundle.json
docs/agentic-self-improvement/eval-report.md
docs/agentic-self-improvement/changelog.md
```

These files should be committed only in self-improvement branches/PRs.
