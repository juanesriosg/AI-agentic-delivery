# Skill: Parallel Agent Execution

Use this skill to safely run multiple agents on the same story.

## Steps

1. Define file ownership and contracts.
2. Split work by specialty.
3. Prevent conflicting edits.
4. Route shared assumptions through the Orchestrator.
5. Merge/reconcile only after tests pass.
6. Log each parallel workstream.

Never let agents independently invent cross-component contracts.
