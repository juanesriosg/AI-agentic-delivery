# Skill: Frontend Engineering

Use this skill when implementing UI or client-side behavior.

## Checklist

- Reuse design system components.
- Keep components small and focused.
- Handle empty, loading, error, success, disabled, and invalid states.
- Use accessible labels and keyboard-friendly controls.
- Align with API contracts.
- Add component/unit tests.
- Add E2E tests for critical journeys.
- Capture visual evidence or prepare Visual QA instructions.
