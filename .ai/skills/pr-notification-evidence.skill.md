# Skill: PR Notification with Evidence

Use this skill when preparing a PR for human AI PM review.

## Required sections

- Story and business goal.
- Summary of code changes.
- Agents involved and iterations.
- QA status and checklist link.
- PM status and checklist link.
- Screenshots/annotations for UI changes.
- Validation commands and results.
- Feedback loop summary.
- Risk and rollback.
- Files worth reviewing carefully.
- Requested human action.
