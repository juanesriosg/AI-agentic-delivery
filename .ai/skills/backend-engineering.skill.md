# Skill: Backend Engineering

Use this skill when implementing server-side behavior.

## Checklist

- Validate inputs.
- Enforce auth/authz.
- Keep business logic testable.
- Use clear errors.
- Add unit and integration tests.
- Add contract tests when API behavior changes.
- Use pagination/limits for collections.
- Add observability where failures need diagnosis.
- Consider idempotency for writes or retries.
