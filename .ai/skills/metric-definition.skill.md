# Skill: Metric Definition

## Purpose

Ensure business metrics are consistently defined before querying data.

## Metric definition template

```md
Metric name:
Business meaning:
Formula:
Population:
Time window:
Grain:
Filters:
Exclusions:
Owner:
Source tables:
Known caveats:
```

## Rules

- Do not answer ambiguous metric questions as if the definition were certain.
- If a metric is not defined, propose a definition and ask for approval.
- If a question can still be answered safely with assumptions, state the assumptions clearly.
- Prefer business-level metrics over technical proxy metrics unless the question is technical.
