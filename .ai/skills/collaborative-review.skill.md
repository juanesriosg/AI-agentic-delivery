# Skill: Collaborative Agent Review

Use this skill when one agent reviews another agent's work.

## Rules

- Review the output against the original spec and your specialty.
- Create feedback with evidence, owner, severity, and verification requirement.
- Do not simply say something is bad; say what is expected, what is actual, and how to verify the fix.
- The implementing agent fixes; the reviewing agent verifies.
- Closure requires a log entry and updated evidence.
