# Skill: Cloud Platform Engineering

Use this skill when cloud, infrastructure, or deployment environments are affected.

## Well-Architected review lens

- Operational excellence: operations as code, telemetry, ownership, rollback.
- Security: least privilege, traceability, data protection, secrets safety.
- Reliability: health checks, failure modes, recovery, backups.
- Performance efficiency: scaling, latency, resource fit.
- Cost optimization: avoid idle resources, right-size non-prod.
- Sustainability: avoid unnecessary resources, prefer efficient managed services.

Production changes require explicit human approval.
