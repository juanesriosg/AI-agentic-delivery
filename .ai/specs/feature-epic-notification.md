# Feature and Epic Completion Notifications

## Feature agent-complete

Use when all tasks for a feature are PR-ready or merged according to repo policy and feature validation is complete or explicitly documented.

```md
Status: Feature agent-complete
Feature:
Epic:
Repos:
Related tasks:
Related PRs:
Validation performed:
Validation gaps:
Risk:
Rollback:
Ready for:
Manager action needed:
```

## Epic agent-complete

Use when all features in the epic are agent-complete and integration-level risk has been assessed.

```md
Status: Epic agent-complete
Epic:
Business outcome:
Completed features:
Related PRs:
Integration validation:
Stakeholder acceptance criteria:
Remaining risks:
Rollout:
Rollback:
Recommended manager review:
```

Data needed:
