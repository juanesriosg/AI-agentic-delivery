# Project Board Lanes

## Backlog

Ideas and work not yet ready for agents.

## Ready for Agent

Tasks with all required fields and `ai:ready`.

## Claimed

Agent has selected the task and branch is being created.

## In Progress

Agent is actively coding.

## Self Testing

Agent is validating and self-reviewing.

## PR Ready

PR is ready and manager has been notified.

## Manager Review

Manager or repo owner is reviewing.

## Changes Requested

Agent needs to respond to review comments.

## QA / Staging

Feature needs integrated validation.

## Done

Manager/repo owner accepted completion.

## Blocked

Task cannot continue without decision, permission, context, or external dependency.

## Recommended labels

